001 — what makes it different

Not a fork. Not a multiplexer.

Three things Halo does that a terminal-plus-tmux stack can't, because Halo isn't running inside another terminal — it is one.

01

It IS the terminal — real libghostty, on the GPU

Halo embeds actual libghostty and renders with Metal. No web view, no shim, no terminal-inside-a-terminal. It reads your real Ghostty config, so any Ghostty key works — theme, font-family, palette, cursor-color, window-padding-x — and colors sync straight from it.

02

A first-class agent-control CLI

Every Halo lives behind a unix socket. The halo CLI lets a script — or an AI agent — both drive and observe terminals: send-keys to type, capture --scrollback to read output back, plus split, focus, zoom, list and reload. Closed loop, not fire-and-forget.

03

Git-worktree-isolated sessions

One call — halo worktree fix/login — spins up an isolated git worktree on its own branch and opens it as a fresh session. Each task gets a clean tree; agents can fan out across branches without stepping on your working copy.

002 — drive & observe

Scriptable
by agents.

The control socket turns a window into an API surface. Point an agent loop at it and it can act, then read what happened — the part most “AI terminal” tools skip.

  • halo split — carve the focused pane
  • halo send-keys focused "…" — type into a pane
  • halo capture --scrollback — read output back
  • halo worktree fix/login — isolated branch session
  • halo list — panes & tabs as JSON
~/Library/Application Support/halo/control.sock 
# an agent investigates a failing test, end to end

Reach for Halo if

  • You drive terminals with AI agents or scripts
  • You want a real GPU terminal, not a layer inside one
  • You run many tasks and want one isolated worktree each
  • You already live in a Ghostty config you'd rather keep

Honestly, skip it if

  • You just want a great terminal to type in by hand
  • You need Linux or Windows — it's macOS-only
  • You need multiple windows today — it's single-window
  • You want signed, polished, stable — this is alpha

If you're not pointing an agent at it, plain Ghostty is the saner choice. Halo wins on native + scriptable + worktree-per-task.

003 — install

Build it from source.

No release binaries yet. You build Halo.app yourself with the bundled script — it's an unsigned alpha, so macOS will ask before opening it the first time.

  1. Clone the repo. You'll need macOS, Xcode command-line tools, and Swift.
  2. Run ./make-app.sh. It builds and bundles Halo.app with libghostty and the fonts.
  3. First launch: right-click → Open (it's unsigned), or clear quarantine, then allow it.
  4. Optional: put the halo CLI on your PATH to drive the running app from scripts and agents.
build

$ cd Halo
$ ./make-app.sh
